Delightful, premium hydration tracking for mobile.

## Features
- **Animated Water Bottle**: A floating, morphing bottle mascot that reacts to your progress.
- **Dynamic Moods**: 4 mood states (Sad, Mild, Okay, Happy) with custom expressions.
- **Persistent Tracking**: Automatically saves your hydration history.
- **Micro-interactions**: Wave effects, bubbles, and premium log toasts.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start the App**
   ```bash
   npx expo start
   ```

## Technology Stack
- **Expo / React Native**
- **React Native Reanimated**: For premium animations.
- **React Native SVG**: For the interactive bottle.
- **AsyncStorage**: For local data persistence.
- **Lucide Icons**: For clean UI elements.
